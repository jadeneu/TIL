### 문제 55 배열 K번째 큰 요소
> 456p

* 정렬되지 않은 배열에서 k번째 큰 요소를 추출하라.
* 입력
  ```
  [3,2,3,1,2,4,5,5,6], k = 4
  ```
* 출력
  ```
  4
  ```
<br><br>

* **내가 짠 코드**<br>
```python
def largest(arr, k) -> int:
  arr.sort()
  return arr[-k]

arr = [3,2,3,1,2,4,5,5,6]
k = 4
print(largest(arr, k))
```
